# 4.能力圈模型

知道自己的能力范围，比单纯的扩大能力范围更重要![截屏2024-10-08 19.03.34](images/%E6%88%AA%E5%B1%8F2024-10-08%2019.03.34.png)

# 9 军标六性

可靠性：稳定完成任务，不掉链子；最基本的

测试性：及时确认状态，隔离内部状态；及时反馈成果，有问题反馈

维修性：具有保持或者恢复功能状态的能力；例如playb，AB岗

适应性：实现预期功能不被破坏；例如，把日常变化纳入计划，在干扰下仍然能够完成任务

安全性：不导致损伤；职场中可以进一步扩展为让合作的人舒服

保障性：相关资源的保障；职场中指的出问题能不能顶上去

# 20.六项思考帽

六类阶段思考法： 

盘控阶段：编排各阶段次序，监控深度和时间，最后推出结论。 

集理阶段：收集整理信息数据，不发表观点。 

萌点阶段：爆发想象、创意、可能性。不论是非对错。 

趋利阶段：寻找价值，建设性观点。 

避害阶段：指出风险，怀疑批判。不谈应对方案。 

抒情阶段：谈主观喜恶，无需理由。

生活中常见的犯错是

1.思考混乱，比如价值和风险比较杂糅；收集到有利信息想优点，收集到不利信息就想缺点

2.漏考虑某些情况

![截屏2025-01-03 14.05.56](images/%E6%88%AA%E5%B1%8F2025-01-03%2014.05.56.png)

所以模型主要是强调思考的几个方向，思考的时候从一个方向考虑，不漏信息

# 21.第一性原则

![截屏2025-01-03 14.07.34](images/%E6%88%AA%E5%B1%8F2025-01-03%2014.07.34.png)

就是数理逻辑中的证明中的基本定理。他是最基础的，是不需要证明的。生活中遇到问题，就是不断从剥离问题，直到找到无法问下的问题，那个就是根。



第一性原理就从根本出发去找到一个新的解决问题的方法，例如火箭是否一定要轨道；这种思考方式有时候能产生颠覆性的效果，但是有时候浪费时间精力；作为下位者的时候，还容易让人觉得无知。



人类文明发展至今，颠覆性的东西也没有那么多。但是我们可以从小的事情上去颠覆，比如开会需要ppt，那么一定是ppt么？本质上是为了方便表达，那思维导图行不行呢？
